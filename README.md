
# Amazon Prime Movies and TV Shows Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Amazon Prime Movies and TV Shows dataset to understand content distribution, genre trends, country-wise production, and release patterns.

The objective of this analysis is to uncover insights about the type of content available on Amazon Prime and how it has evolved over the years.

## Dataset
The dataset contains information about movies and TV shows available on Amazon Prime including:
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Data Transformation
5. Exploratory Data Analysis
6. Visualization and Insights

## Key Analysis Performed
- Movies vs TV Shows distribution
- Most common genres
- Content distribution by country
- Release year trends
- Ratings distribution
- Top contributing directors and actors

## Key Insights
- Movies dominate the platform compared to TV shows.
- A few genres account for a large portion of the content.
- The number of titles added has significantly increased in recent years.
- Certain countries contribute more content to the platform.

## Repository Structure
- `amazon_prime_analysis.ipynb` – Jupyter Notebook containing the analysis
- `dataset.csv` – Dataset used for analysis
- `visualizations/` – Charts generated during analysis

## Conclusion
The analysis provides valuable insights into Amazon Prime's content library, highlighting trends in genre popularity, content growth, and geographical distribution.
